# Layali Café & Desserts Website

## About the Project
This project is a responsive website for **Layali**, a local café in **Magharibi, Kenya**. The website is designed to promote the business and provide information about its products and services.

Layali specializes in:
-  Milkshakes
-  Mojitos
-  Waffles
-  Mug Cookie Crumbs

## Objectives
- Create an online presence for Layali.
- Showcase the café's products and services.
- Develop a responsive and user-friendly website.
- Apply web development skills using HTML, CSS, JavaScript, and Bootstrap.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Git and GitHub

## Planned Pages
